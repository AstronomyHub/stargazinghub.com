# 天文通 (Tianwentong) 功能说明

> 天文通是一款专为天文爱好者和观星者设计的应用程序，提供天文事件预报、星图查看、天文知识学习等功能。

---

## 📱 主要功能模块

### 🔭 天文观测

| 功能 | 说明 |
|------|------|
| **星图 (Star Chart)** | 交互式星图，实时显示星空位置，支持缩放和平移（非简体中文语境只要简单提） |
| **极光 (Aurora)** | 极光预报和实时监测，显示极光概率和强度 |
| **月球位置 (Moon Locations)** | 月球表面特征位置标注和导航 |
| **银河 (Milky Way)** | 银河可视化和光污染地图叠加 |
| **观星指数 (Stargazing Index)** | 综合评分评估当晚观星条件 |

### ☀️ 太阳系天体

| 功能 | 说明 |
|------|------|
| **太阳 (Sun)** | 太阳活动监测和太阳黑子信息 |
| **月球 (Moon Phase)** | 月相日历，显示每日月相和升起落下时间 |
| **月食 (Lunar Eclipse)** | 月食时间预报和观测指南 |
| **木星 (Jovian)** | 木星及其卫星实时位置和天体事件 |
| **彗星 (Comets)** | 亮彗星预报和详细信息 |
| **行星位置 (Planet Positions)** | 行星实时位置和星历表 |

### 🌠 流星雨

| 功能 | 说明 |
|------|------|
| **流星雨日历 (Meteor Shower Events)** | 年度流星雨时间表和预报 |
| **流星雨图表 (Meteor Shower Chart)** | ZHR 变化曲线和最佳观测时间 |
| **流星雨详情 (Meteor Shower Hourly)** | 每小时流星数量预测 |
| **历史数据 (Meteor History)** | 历史流星雨观测记录 |
| **流星事件 (Meteor Events)** | 重要流星事件提醒 |

### 🛰️ 卫星追踪

| 功能 | 说明 |
|------|------|
| **星链 (Starlink)** | 星链卫星过境预报和实时追踪 |
| **卫星过境 (Satellite Pass)** | 人造卫星过境时间和位置预测 |

### 🌤️ 天气预报

| 功能 | 说明 |
|------|------|
| **15天预报 (15-Day Forecast)** | 长期天气预报 |
| **小时预报 (Hourly Forecast)** | 逐小时天气预报 |
| **卫星云图 (Cloud Cover)** | 云量监测和预报 |
| **雷达 (Radar)** | 天气雷达图像 |
| **空气质量 (Air Quality)** | 空气质量指数监测 |
| **天文天气预报 (7Timer)** | 专为天文观测优化的天气预报 |

### 🌌 深空观测

| 功能 | 说明 |
|------|------|
| **星空指南 (Sky Guide)** | 每日推荐观测目标和信息 |
| **天文画廊 (Astronomy Gallery)** | 天文摄影作品展示 |
| **光污染地图 (Light Pollution Map)** | 附近光污染等级查询 |

### 📸 天文摄影

| 功能 | 说明 |
|------|------|
| **曝光计算器 (Exposure Calculator)** | 行星摄影曝光参数计算 |
| **导星精度计算器 (Guiding Accuracy)** | 导星精度评估和优化建议 |
| **等效焦距 (Equivalent Focal)** | 相机镜头等效焦距计算 |

### 🔧 实用工具

| 功能 | 说明 |
|------|------|
| **极轴镜 (Polar Scope)** | 极轴镜校准和设置向导 |
| **红光模式 (Red Light)** | 护眼红光模式，保护暗适应 |
| **天文事件 (Astronomical Events)** | 日月食、行星凌日等重大事件列表 |
| **单位设置 (Unit Settings)** | 单位制自定义（公制/英制） |
| **无线电流星监测 (Radio)** | 射电天文相关功能 |
